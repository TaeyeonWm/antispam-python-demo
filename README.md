- 文件说明

```

.
├── audio 点播语音接口演示
├   │── audio_callback.py 点播语音结果获取
├   │── audio_check.py 点播语音提交
├── liveaudio 直播语音接口演示
├   │── liveaudio_callback.py 直播语音结果获取
├   │── liveaudio_check.py 直播语音提交
├── filesolution 文档解决方案接口演示
├   │── filesolution_callback.py 文档解决方案结果获取
├   │── filesolution_submit.py 文档解决方案提交
├── videosolution 点播音视频解决方案接口演示
├   │── videosolution_callback.py 点播音视频解决方案结果获取
├   │── videosolution_submit.py 点播音视频解决方案提交
├── image 图片接口演示
│   ├── callback.py
│   └── check.py
├── video
│   ├── livevideo_callback.py 直播视频流结果获取
│   ├── livevideo_check.py 直播视频流提交
│   ├── video_callback.py 视频结果获取
│   └── video_check.py 视频流提交
├── README.md
└── text 文本接口演示
    ├── callback.py
    └── check.py
```
